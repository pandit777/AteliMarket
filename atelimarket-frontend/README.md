> Smart Shopping, Smarter Savings 🚀

## ✨ Features

- 🛍️ **Product Management** - Add, edit, delete products with images
- 🛒 **Shopping Cart** - Add items, update quantities, calculate totals
- 💬 **WhatsApp Integration** - Direct order confirmation via WhatsApp
- 🔐 **Admin Panel** - Secure authentication with JWT
- 📸 **Image Upload** - Product images with Multer
- 🎨 **Modern UI** - Tailwind CSS, responsive design
- 🔍 **Search & Filter** - Search by name, category, or price
- 📊 **Stock Management** - Track product inventory

## 🚀 Tech Stack

### Frontend
- **React 18** - UI Framework
- **Tailwind CSS** - Styling
- **React Router v6** - Navigation
- **Axios** - API calls

### Backend
- **Node.js** - Runtime
- **Express** - Web framework
- **MongoDB Atlas** - Database
- **JWT** - Authentication
- **Multer** - File upload
- **Bcrypt** - Password hashing
